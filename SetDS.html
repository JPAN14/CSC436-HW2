This is an implementation of the Set Data Structure using JavaScript

Implementation was modified from https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Set

Look in the console to see the output.

I implemented the Set as follows:

  • var set = new Set();
  • set.add(1);
  • set.add(10);
  • set.add("here is some text");
  • var o = {a: 1, b: 2};
  • set.add(o);

Following which, there are a bunch of tests to see whether the Set contains certain elements.

  • set.has(1); This will return true.
  • set.has(3); This will return false.
  • set.has(10); This will return true.
  • set.has(Math.sqrt(100)); This will return true as well.
  • set.has("Here is some text".toLowerCase()); This will also return true.
  • set.has(o); This will return true.
  • set.size; This will return 4.

Another element is removed as follows:

  • set.delete(10); This will delete the element 10 from the Set

Followed by the final 2 tests

  • set.has(10); This will return false.
  • set.size; This will return 3.
